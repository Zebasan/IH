Listor i HTML

1. Oordnad lista (punktlista)

För att skapa en punktlista använder man taggen <ul>, och varje punkt skapas med <li>.

<ul>
    <li>Äpple</li>
    <li>Banan</li>
    <li>Apelsin</li>
</ul>
  • Äpple
  • Banan
  • Apelsin

2. Ordnad lista (numrerad)

Skapas med <ol> och <li>.

<ol>
    <li>Förbered ingredienser</li>
    <li>Värm ugnen</li>
    <li>Grädda kakan</li>
</ol>
  1. Förbered ingredienser
  2. Värm ugnen
  3. Grädda kakan

3. Nästlad lista (lista i lista)

<ul>
    <li>Dator</li>
    <li>Tillbehör
        <ul>
            <li>Mus</li>
            <li>Tangentbord</li>
        </ul>
    </li>
    <li>Skärm</li>
</ul>
  • Dator
  • Tillbehör
    • Mus
    • Tangentbord
  • Skärm

Formulär och Inputfält i HTML

1. Grunderna i ett formulär

För att skapa ett formulär använder man taggen <form>. Inuti formuläret använder man <input> för att lägga till fält, och <button> eller <input type="submit"> för att skicka formuläret.

Exempel på HTML-kod:

<form action="#" method="post">
    <label for="namn">Namn:</label>
    <input type="text" id="namn" name="namn"><br><br>

    <label for="epost">E-post:</label>
    <input type="email" id="epost" name="epost"><br><br>

    <label for="lösenord">Lösenord:</label>
    <input type="password" id="lösenord" name="lösenord"><br><br>

    <input type="submit" value="Skicka">
</form>

2. Andra typer av inputfält

Du kan använda olika typer av <input> för olika syften, t.ex. nummer, datum, kryssrutor eller radioknappar.

Exempel på HTML-kod:

<input type="number">
<input type="date">
<input type="checkbox">
<input type="radio">

3. Textarea och knapp

<textarea> används för att skriva längre meddelanden. <button> kan användas som alternativ till input type="submit".

Exempel på HTML-kod:

<textarea rows="4" cols="30">Skriv något här...</textarea>
<button>Skicka</button>
